# ONI-DrainValve
description:
• DrainValve - Custom Drain Amount for Steam Turbine.

Features:
• Adds a configurable drain valve to Steam Turbine.
• Solves the "water swallowing / stuck water" problem of Steam Turbine.

How to Use:
• The configuration panel defaults to 2kg per drain.
• You can set decimal values with the slider (use values below 2kg with caution).

Notes:
• If multiple Steam Turbines share the same liquid pipe, it is recommended to keep the default value. Otherwise, the turbines may stop due to pipe blockage.

Known Issues:
• Causes approximately 3% performance loss.

Credits:

中文版：

功能特性：
• 为蒸汽涡轮机（Steam Turbine）添加了可调节的排水阀；
• 彻底解决蒸汽涡轮机“吞水”/积水卡住的问题。

使用方法：
• 配置面板默认每次排放 2kg，目前只能输入整数，滑块支持小数点调节；
• 请谨慎使用低于 2kg 的数值。

注意事项：
• 如果多台蒸汽涡轮机共用同一条排水管，建议保持默认值，否则可能因水管堵塞导致蒸汽机停机。

已知问题：
• 会造成约 3% 的性能损失。

• 配置界面基于 PeterHan 的 PLib 实现（MIT License）。
